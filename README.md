# Event & Activity Management System

## Overview
Developed a full-stack event management system using PHP, Node.js, MySQL, and Firebase. Designed interfaces with Vue.js and Bootstrap. Enhanced event registration and workflow efficiency.

## Features
* User Authentication and Authorization: Secure login and role-based access control for different user types.
* Event Creation and Management: Easy-to-use interface for creating new events, setting dates, locations, and descriptions.
* Participant Registration: Users can easily browse available events and register to participate.
* Dashboard Panels: Dedicated dashboards for each role to track progress, monitor attendance, and manage tasks.
* Responsive Design: The web interface is responsive and accessible across different devices.

## Technologies Used
* Backend: PHP, Node.js
* Database: MySQL, Firebase
* Frontend: Vue.js, Bootstrap

## Setup Instructions
1. Clone the repository:
   git clone https://github.com/lamahashim1/EventManagementSystem.git

2. Configure your local PHP/Node.js environment and connect to your MySQL/Firebase databases as per the configuration files.
